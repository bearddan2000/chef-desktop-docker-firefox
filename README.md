# chef-desktop-docker-firefox

## Description
POC project to show chef built app.

## Tech stack
- chef 15
- ruby 2.7
- firefox
- bash

## Docker stack
- docker-cli
- ubuntu:latest

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
